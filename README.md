# 💻 Operating System Lab & Concepts

## 📌 Overview

This repository contains practical implementations, experiments, and study materials related to Operating System concepts.

The goal of this repository is to understand how operating systems manage processes, memory, files, CPU scheduling, synchronization, and system resources through hands-on programming and simulations.

---

## 🎯 Objectives

- Understand Operating System fundamentals
- Learn process management
- Explore CPU scheduling algorithms
- Study memory management techniques
- Understand synchronization concepts
- Learn file system operations
- Gain practical experience with OS algorithms

---

## 📚 Topics Covered

### Process Management
- Process Creation
- Process States
- Process Control Block (PCB)
- Context Switching

### CPU Scheduling Algorithms
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin Scheduling

### Memory Management
- Paging
- Segmentation
- Virtual Memory
- Page Replacement Algorithms

### Process Synchronization
- Critical Section Problem
- Semaphores
- Mutex Locks
- Producer Consumer Problem

### Deadlocks
- Deadlock Prevention
- Deadlock Avoidance
- Banker's Algorithm

### File Management
- File Operations
- Directory Structures
- File Allocation Methods
